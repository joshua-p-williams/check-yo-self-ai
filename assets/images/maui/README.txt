MAUI-first source assets for CheckYoSelfAI

Files
-----
appiconfg.svg  -> foreground mark only, transparent background
appiconbg.svg  -> optional solid background artwork
splash.svg     -> centered mark for splash usage
theme.txt      -> theme colors

Suggested MAUI usage
--------------------
App icon foreground:
  assets/images/check-yo-self-icons/maui-source/appiconfg.svg

App icon background:
  Either use Color="#0B3C5D" on <MauiIcon ...>
  or use appiconbg.svg if you prefer artwork background.

Splash:
  assets/images/check-yo-self-icons/maui-source/splash.svg
  Background color: #0B3C5D

Notes
-----
- These are source assets, not platform packs.
- SVG is preferred for MAUI.
- If you want pixel-exact refinement, open these in Figma or Inkscape.
